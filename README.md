# Przelicznik
Przelicza stopnie Celsjusza na stopnie Fahrenheita
oraz złotówki na dolary.

# Jak uruchomić?
Program można uruchomić korzystając z pliku przelicznikGUI.exe,
lub uruchamiając skrypt przelicznikGUI.py\
Wybierając drugą opcję należy spełniać jednak pewne wymagania podane niżej.

# Wymagania:
Python 3.5 oraz moduły:\
  -Pillow\
  -requests\
Oba moduły można zainstalować wpisując w oknie konsoli komendę\
pip install [nazwa modułu]
